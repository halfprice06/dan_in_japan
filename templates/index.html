Dan in Japan ダンイン ジャパン

In October 2024, I spent 2 weeks in Japan with my wife Christina and our good friends Chuck and Ashley. We visited Tokyo, Osaka, and Kyoto, taking nearly 2000 photos along the way.

After manually filtering out duplicates and not so great photos, I built an interactive gallery using the SerpAPI and Claude 3.5 Sonnet to auto-generate captions that are photo metadata aware! The system also uses the Instructor library to extract interesting cultural and historical points from each image, making it easy to learn more about what you're seeing. The AI captions are not perfect, but they're a fun way to learn more about the photos.

For more details about how the AI captioning works, check out the about page. The full code is available on GitHub.
